* DeepaMehta 4 Tags Plugin 1.2 Re-Release

Plugin Direct Download:
http://download.deepamehta.de/nightly/deepamehta-tags-1.2.jar

Note: A "Tag"-Type is not assigned to any other type by default, so you
have to interactively enable "Tagging" for e.g. the Type "Note" or the
Type "Web Resource" (build up "Aggregation Definition" with "Many"
"Tags". This is on purpose, though I am considering of enriching the
"Web Resource" about "Tags" by default (during installation of this
"deepamehta-tags"-Module) so that users of the next generation of the
"DeepaMehta Firefox Toolbar" don't need to configure something manually
after installation of the Tags-Module but are ready to go.

Detailed configuration hints are available at:
https://github.com/mukil/dm4.tags#deepamehta-4-tags-module

Bundled for download and thanks to jri's investigations of the issue [1]
now fully compatible with any existing DeepaMehta 4 installation.

Plugin Direct Download:
