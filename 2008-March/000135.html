For the moment non-administrator users are not able to maintain 
their profile themselfs. That is a user is not able to edit the 
"Description" field of its own User-Topic (same applies to the 
"Username", "Password", "First Name", "Last Name", "Gender", 
"Birthday", and "Icon" fields).

In order to strengthen DeepaMehta's social software aspect, every 
user should be able to edit the following fields of _its own_ User-Topic:
- "First Name", "Last Name"
- "Gender", "Birthday"
- "Description"
- "Icon"
- "Password"

Note: the "Username" field should only editable for administrators.
