Dear Developers,

we have a new DeepaMehta developer: Urs Lerch from Switzerland.  
Currently he is doing his PhD about open source and would like to use  
his knowledge in a practice-based open source project. From now on he  
has SVN write permission.
www.deepamehta.de/docs/humans.html#friends

His first tasks are
1) adding a LICENSE file. The upcoming DeepaMehta 2.0b8 release will  
use the GPL 3 license
2) sourcecode maintenance regarding Java Collections Framework. See bug  
report:
https://developer.berlios.de/bugs/? 
func=detailbug&bug_id=14005&group_id=5174
3) Quality assurance (nightly builds). Installation tests on various  
platforms and database systems

He is also interested to foster the DeepaMehta community process and to  
network the DeepaMehta project with other Open Source projects. He is  
the guy who e.g. initiated DeepaMehta's participation with the Google  
Summer of Code event.

Welcome Urs!

Jörg R.
