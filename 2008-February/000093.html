Hello Malte,

about a year ago I resolved a Windows installer issue by using the great 
Nullsoft installer framework which already has been released on sourceforge:

http://nsis.sourceforge.net/Main_Page

Nsis is a scriptable framework. So you write one or more plain text ini 
scripts defining the source files, the destination, welcome screens and 
stuff (even advanced things like checking settings) and compile the 
packages afterwards. Interesting for you might be that you can compile 
windows packages with nsis on a linux (maybe even os x?) computer.

Maybe give it a try.
