DeepaMehta provides a demo mode to demonstrate its features to 
anyone. Demo users don't need a DeepaMehta account and can 
navigate through a prepared demo topic map. Demo users are not 
able to madify any content. Currently in demo mode the "Search" 
command is not available (only "What's related?" is provided). 

Feature Request: The "Search" command should be availale in demo 
mode and should provide a list of all topic types which are assigned to 
the workspace the demo topic map originates from.
